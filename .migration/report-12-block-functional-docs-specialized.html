16. Mondrian

4 VariationsComplexity: ComplexEst: 4–6 dev daysPriority: P2
Block Overview: Multi-slot promotional module using a Mondrian-style grid layout. Features large and small image tiles, CTA links, and optional inline navigation lists. Two primary structural variants (module-d, module-f) with CSS-only sub-variants. Found on Legacy platform category hub pages.

V1: Module-D (Large Left + Small Right)

Legacy

Two-column layout: left side has a large hero image slot (376×170px), right side stacks a small image tile (174×170px) above a navigation list. Used on L1 category hub pages.

Structure / Fields

ElementTag/ClassDescription
Container.mondrian-module.mondrian-module-d.clearfixModule-D wrapper (724×170px)
Left Panel.mondrian-module-outer.mondrian-submodule-leftLarge image slot (376×170px) with link + overlay text
Right Top.mondrian-module-innerSmall image tile (174×170px) with link + overlay
Right Bottom.mondrian-module-inner.mondrian-module-listNavigation link list (no image)

Acceptance Criteria

  • Grid proportions: left ~52%, right-top ~24%, right-bottom ~24%
  • Left panel image fills slot without distortion
  • Right-top image tile displays with overlay text
  • Right-bottom renders as navigation link list
  • Nav items: Cell & Gene Therapy, Clinical Microbiology, Diagnostic Development, Diagnostic Testing
  • All tiles/links are clickable and navigate correctly
  • Clearfix prevents float collapse

Interactions

Hover (Tiles)
Image tiles: slight zoom (scale 1.05) + overlay darkening. Nav links: underline + color shift.
Click
Each tile/link navigates to category page.

Reference URLs

clinical.html

  • Dark overlay covers page background (rgba(0,0,0,0.7))
  • Close button (X) in top-right corner

  • Focus trapped within modal when open
  • Escape key closes modal
  • Focus returns to trigger element on close
  • aria-modal="true" on modal container
  • role="dialog" with aria-label
